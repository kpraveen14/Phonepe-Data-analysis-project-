## ğŸ“ˆ Key Insights & Visuals
Key Insights Identified
- Digital transactions show strong month-on-month growth, especially in Tier 1 & Tier 2 cities.

- UPI payments dominate, contributing major volume in both P2P and P2M segments.

- Certain states (e.g., Karnataka, Maharashtra, Uttar Pradesh) show highest transaction density.

- Merchant categories like recharge, utilities, and retail payments are rapidly expanding.

- Peak transaction times cluster around evenings and weekends, indicating user behavior patterns.

- Transaction Volume Trend (Line Chart)

- State-wise Usage Heatmap

- Category-wise Payment Breakdown

- Top Cities by Transaction Value

- Monthly Growth KPIs

- Filters for Region, Category, and Transaction Mode
